- Langage : Python
- Données : Pandas
- API & streaming : API Interactive Brokers, WebSocket
- Outils : IBridgePy, Tkinter

### Réalisations clés

- Analyses techniques et fondamentales pour identifier des opportunités de trading
- Stratégies de trading basées sur les données de marché et le contexte macroéconomique
